## Installation
1. clone the project using git 
```bash
git clone https://github.com/neoScriptscode/sqlite_crud.git
```
2. cd into project dir
```bash
cd sqlite_crud
```

3. run 

```bash
npm i
```

```bash
npm run dev
```
